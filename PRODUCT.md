# Product

## Register

product

## Users

Developers managing Git repositories in a focused desktop workflow. They need repository state, diffs, history, and write operations to remain fast and understandable while moving between keyboard and mouse input.

## Product Purpose

rgitui is a GPU-accelerated desktop Git client that makes everyday and advanced Git workflows responsive, direct, and visually clear across repositories and platforms.

## Brand Personality

Fast, precise, native. The interface should feel dependable and focused, with its visual system serving the repository task rather than drawing attention to itself.

## Anti-references

Avoid ornamental web or generic SaaS styling, decorative interaction, unfamiliar controls, and presentation that obscures repository state or slows down common workflows.

## Design Principles

- Keep the user in flow with responsive operations and useful prefetching.
- Make availability and state obvious before the user acts.
- Preserve familiar desktop and Git interaction patterns.
- Support equivalent, discoverable keyboard and mouse workflows.
- Fit improvements into the existing product language instead of redesigning isolated surfaces.

## Accessibility & Inclusion

No formal conformance target is currently required. Maintain strong contrast across themes, clear focus and disabled states, understandable status feedback, and practical keyboard and mouse parity. Avoid relying on color alone to communicate state.
